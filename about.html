课程理念

用代码点亮思维，以创造丈量未来

🧠

计算思维优先

不止于语法，更注重培养分解问题、识别模式、抽象建模的能力，让编程成为解决真实问题的工具。

🎮

项目驱动学习

每一节课都有完整的项目成果——一个游戏、一个工具、一个作品。孩子们在"造东西"的过程中真正理解编程。

🌱

低门槛、高天花板

以 Scratch 图形化积木起步，逐步过渡到 Python 文本编程，分层设计保证每个层次的孩子都能找到成就感。

课程体系

PHASE · 01
启蒙探索阶段 / Scratch 入门
通过拖拽积木完成趣味动画与小游戏，建立"程序 = 逻辑指令"的直觉认知，培养序列、循环、条件三大编程基础概念。
Scratch动画循环条件判断
PHASE · 02
进阶创作阶段 / 图形化 → 文本衔接
引入 MBlock，让积木背后的 Python 代码"现身"，让孩子自然感知图形化与文本编程的对应关系，降低语法焦虑。
MBlockPython对照函数变量
PHASE · 03
项目实战阶段 / Python 独立编程
完全转向 Python 文本编程，综合运用列表、字典、函数等知识，独立完成具有实用价值的完整项目。
Python数据结构项目制调试技巧
PHASE · 04
创新拓展阶段 / AI & 硬件应用
接触机器学习 API 与 micro:bit 硬件编程，把代码从屏幕延伸到现实世界，感受科技改变生活的力量。
AI应用micro:bit物联网创客

常见问题

孩子没有任何编程基础，能学会吗？▼
完全没问题！课程从最基础的 Scratch 图形化积木起步，不需要任何先验知识。只要孩子对"创造"和"游戏"感兴趣，就能在这里找到属于自己的节奏。
课程资源可以下载供孩子课后复习吗？▼
可以！每节课的源程序、课件和素材包都可以在"课程首页"对应的课程详情页中下载。部分课程资源还在持续整理中，会陆续更新。
孩子的作品如何提交展示到"学生作品"页面？▼
将作品截图/演示视频发送给老师，老师会在审核后添加到作品展示页。优秀作品还有机会参与校内、市级比赛推荐！
课程用什么软件？需要提前安装吗？▼
主要使用 MBlock 5（含 Scratch 3 模式和 Python 模式），官网免费下载，也支持网页版在线使用，无需安装。部分课次会使用 Python IDLE 或 Thonny。老师会在课前提前告知并指导安装。
